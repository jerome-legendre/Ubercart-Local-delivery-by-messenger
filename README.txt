Ubercart Local delivery by messenger
=====================================
This shipping module add checkout pane for delivery time choosing .

Administration page is
admin/store/settings/quotes/uc-messenger

Settings are:

* List of Postal code accepted (delivery address),
* Rate (percentage +/-),
* Preparation and delivery time in minutes,
* For each time slot:
* 		operating days
* 		time slot begin and end
*	 	Minimum purchase amount required




[FRENCH]

Ubercart module de Livraison par coursier
========================================

Ce module apporte un plugin de livraison Ubercart dédié à la livraison locale ty nourriture livrée par coursiers


Administration à la page
admin/store/settings/quotes/uc-messenger


Vous pouvez régler sur la page d'administration:

* les codes postaux acceptés en adresse de livraison
* le cout (+/-) en pourcentage applique au mode de livraison, 
* le délai minimum avant  livraison (temps de préparation et de transport), 
* Pour chacun des créneaux horaires:
*		les jours ouverts
*		l'heure de début et de fin du créneau
*	 	le minimum de commande accepté
